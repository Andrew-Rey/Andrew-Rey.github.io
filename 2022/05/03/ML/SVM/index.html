т╝ЋУеђ

Т▓АС╗ђС╣ѕт╝ЋУеђ, тДЉСИћУ░ѕУ░ѕС╗ђС╣ѕТў» тЄйТЋ░жЌ┤жџћ, С╗ђС╣ѕТў» тЄаСйЋжЌ┤жџћ тљД.

тЁХт«ътцДжЃетѕєТЮЦУЄфС║јуЏ┤УДЅтњїУДѓт»ЪтљД. Т»ћтдѓСйауюІСИІжЮбУ┐Ўт╣ЁтЏЙ, тЦйуюІСИЇ? ТѕЉућ╗уџё.

тЂЄУ«ЙУЊЮУЅ▓уѓ╣СИ║ТГБСЙІ, ТўЙуёХуЏ┤у║┐С╗БУАеуџёТў»тѕєуЋїжЮб, т╣ХСИћС╗јуЏ┤УДѓСИіТёЪтЈЌ: т»╣С║јТЪљСИфУЊЮУЅ▓уѓ╣УђїУеђ, уд╗тѕєуЋїжЮбУХіУ┐ю, т«ЃТў»ТГБСЙІуџётЈ»УЃйТђДУХіжФў, Уђї ТЪљСИфУЊЮУЅ▓уѓ╣уд╗тѕєуЋїжЮбУХіУ┐Љ, т«ЃТў»ТГБСЙІтЈ»УЃйТђДУХіт░Ј (тЇ│УбФУ»»тѕцуџётЈ»УЃйТђДУХіжФў). тЏаТГц, УЃйСИЇУЃйСй┐ућеСИђуДЇТќ╣Т│Ћ, ТѕќУђЁТў»УАеУЙЙТќ╣т╝ЈТЮЦт▒Ћуц║У┐ЎуДЇуЏ┤УДЅтЉб? уГћТАѕТў» УиЮуд╗.

СИђУѕгУђїУеђ, СИђСИфуѓ╣УиЮуд╗тѕєуд╗УХЁт╣│жЮбуџёУ┐юУ┐ЉтЈ»С╗ЦУАеуц║тѕєу▒╗жбёТхІуџёуА«С┐АуеІт║д. У«Йтѕєуд╗УХЁт╣│жЮбСИ║ \(w^Tx+b=0\), тѕЎ \(|w^Tx+b|\) УАеуц║С║єТЪљСИфТаиТюгуѓ╣ \(x\) СИјУХЁт╣│жЮбуџёУиЮуд╗. Уђї \(|w^Tx+b|\) уџёугдтЈиСИју▒╗ТаЄуГЙ \(y\in{-1,1}\) Тў»тљдСИђУЄ┤УЃйтцЪУАеуц║тѕєу▒╗Тў»тљдТГБуА«. ТЅђС╗Ц, уће \[ y(w^Tx+b) \] тЈ»С╗ЦУАеуц║тѕєу▒╗уџёТГБуА«ТђДтњїуА«С┐Ат║д. У┐ЎСЙ┐Тў»ТЅђУ░ЊуџётЄйТЋ░жЌ┤жџћ.

тЄйТЋ░жЌ┤жџћ

У«Гу╗ЃжЏє \(T\), УХЁт╣│жЮб \((w, b)\), \((w, b)\) тЁ│С║јТЪљТаиТюгуѓ╣ \((x_i, y_i)\) уџётЄйТЋ░жЌ┤жџћт«џС╣ЅСИ║ \[ \hat{\gamma}_i=y_i(w^Tx_i+b) \]

УђїУХЁт╣│жЮб \((w, b)\) тЁ│С║ј \(T\) уџётЄйТЋ░жЌ┤жџћт«џС╣ЅСИ║ \[ \hat{\gamma}=min_{i=1,\cdots,n}\hat{\gamma}_i \]

УђЃУЎЉтѕ░ \(\omega\) тњї \(b\) Тў»жйљТгАуџё, тљїТЌХтбътцДТѕќтЄЈт░ЈТЌХС╝џтй▒тЊЇтѕ░тЄйТЋ░жЌ┤жџћ, тЏаТГцт╝ЋтЄ║тЄаСйЋжЌ┤жџћуџёТдѓт┐х:

тЄаСйЋжЌ┤жџћ

У«Гу╗ЃжЏє \(T\), УХЁт╣│жЮб \((w, b)\), \((w, b)\) тЁ│С║јТЪљТаиТюгуѓ╣ \((x_i, y_i)\) уџётЄйТЋ░жЌ┤жџћт«џС╣ЅСИ║ \[ \gamma_i=y_i(\frac{w^T}{||w||}x_i+\frac{b}{||w||}) \]

УђїУХЁт╣│жЮб \((w, b)\) тЁ│С║ј \(T\) уџётЄйТЋ░жЌ┤жџћт«џС╣ЅСИ║ \[ \gamma=min_{i=1,\cdots,n}\hat{\gamma} \]

ТГБжбў

уАгжЌ┤жџћу║┐ТђДSVM

SVMУдЂТ▒ѓТюђтцДтїќтЄаСйЋжЌ┤жџћ, УЃйСй┐УХЁт╣│жЮбС╗ЦТюђтцДуџёуй«С┐Ат║дт░єТаиТюгтѕєу▒╗ \[ \begin{aligned} max_{w,b}\ \ &\gamma \\ s.t.\ \ &y_i(\frac{w^T}{||w||}x_i+\frac{b}{||w||}) \geq \gamma,\ \ i=1,\cdots,n \end{aligned} \]

т░єтЄаСйЋжЌ┤жџћтїќСИ║тЄйТЋ░жЌ┤жџћ, \(\gamma=\frac{\hat{\gamma}}{||w||}\), т╣ХУђЃУЎЉтѕ░тЄйТЋ░жЌ┤жџћСИј \((w,b)\) ТЌатЁ│, тЈќ \(\hat{\gamma}=1\), тЙЌтѕ░тдѓСИІС║їТгАУДётѕњ: \[ \begin{aligned} min_{w,b}\ \ &\frac{1}{2}||w||^2 \\ s.t.\ \ &y_i(w^Tx_i+b) \geq 1,\ \ i=1,\cdots,n \end{aligned} \]

С╝ўтїќУ»ЦжЌ«жбў:

\(Lagrange\) тЄйТЋ░: \[ L(w,b,\lambda)=\frac{1}{2}w^Tw+\sum_i\lambda-\sum_i\lambda_i y_i(w^Tx_i + b) \] т»╣тЈѓТЋ░Т▒ѓТб»т║д: \[ \begin{aligned} \frac{\partial}{\partial w}L&=w-\sum_i\lambda_i y_i x_i = 0 &\qquad(1)\\ \frac{\partial}{\partial b}L&=-\sum_i\lambda_i y_i = 0 &\qquad(2) \end{aligned} \] тИдтЁЦ \(Lagrange\) тЄйТЋ░тЙЌт»╣тЂХтЄйТЋ░ (тЇ│ТІЅТа╝ТюЌТЌЦтЄйТЋ░тЁ│С║јтЈѓТЋ░уџёТъЂт░Јтђ╝): \[ -\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j + \sum_i\lambda_i \] тЙЌтјЪжЌ«жбўуџёт»╣тЂХжЌ«жбў: \[ \begin{aligned} max_{\lambda}\ \ &-\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j + \sum_i\lambda_i \\ s.t.\ \ &\sum_i\lambda_i y_i = 0 \\ &\lambda\geq 0 \end{aligned} \] Тѕќ: \[ \begin{aligned} min_{\lambda}\ \ &\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j - \sum_i\lambda_i \\ s.t.\ \ &\sum_i\lambda_i y_i = 0 \\ &\lambda\geq 0 \end{aligned} \] ућ▒ \(KKT\) ТЮАС╗Х: \[ \begin{aligned} \nabla_{w;b}L&=0 \\ \lambda_i&\geq 0 \\ y_i(w^Tx_i+b) &\geq 1,\ \ i=1,\cdots,n \\ \lambda_i(y_i(w_*^Tx_i + b_*) - 1) &= 0 \end{aligned} \] УђЃУЎЉтѕ░ \(y_i^2=1\) тЙЌ: \[ \begin{aligned} w_*&=\sum_i\lambda_i y_i x_i \\ b_*&=y_i-\sum_i \lambda_i y_i x_i^Tx_j \end{aligned} \] ТГцтцёуџё \(\lambda_i\) УАеуц║ТюђС╝ўуџё \(\lambda_*\).

тЏаТГцтЙЌтѕ░тѕєуд╗УХЁт╣│жЮб: \[ y(x) = (\sum_i\lambda_i y_i x_i)^Tx + y_i-\sum_i \lambda_i y_i x_i^Tx_j \] ТГцтцёуџё \(\lambda_i\) УАеуц║ТюђС╝ўуџё \(\lambda_*\).

Тюђтљјуџётє│уГќтЄйТЋ░тЈ»С╗Цт«џС╣ЅСИ║: \[ f(x) = sgn (y(x)) \]

ТЮЦСИфСЙІжбў: ућеС║їТгАУДётѕњтњїт»╣тЂХжЌ«жбўСИцуДЇТќ╣т╝ЈТ▒ѓУДБ.

уГћТАѕ: \[ \frac{1}{2}x^{(1)}+\frac{1}{2}x^{(2)}-2=0 \]

Уй»жЌ┤жџћу║┐ТђДSVM

ТЋ░ТЇ«жЏєу║┐ТђДСИЇтЈ»тѕє, У«Йуй«ТЮЙт╝ЏтЈўжЄЈ \(\xi\). у║┐ТђДСИЇтЈ»тѕєУАеуц║жЃетѕєТЋ░ТЇ«СИЇУЃйТ╗АУХ│тЄйТЋ░жЌ┤жџћтцДС║јуГЅС║ј1. СйєТѕЉС╗гтИїТюЏТи╗тіаТЮЙт╝ЏтЈўжЄЈтљјтЈ»С╗ЦТ╗АУХ│. \[ y_i(w^Tx_i+b)\geq 1-\xi_i \] СйєТѕЉС╗гСИЇтЁЂУ«ИТЌажЎљтѕХтю░т╝ЋтЁЦТЮЙт╝ЏтЈўжЄЈ, тЏаТГцжюђУдЂУ«Йуй«СИђт«џуџёС╗БС╗и: \[ C\sum_i\xi_i,\ \ C > 0 \] тЏаТГцтјЪжЌ«жбўуГЅС╗иС║јСИІУ┐░С║їТгАУДётѕњ: \[ \begin{aligned} min_{w,b}\ \ &\frac{1}{2}||w||^2 + C\sum_i\xi_i \\ s.t.\ \ &y_i(w^Tx_i+b) \geq 1 - \xi_i,\ \ i=1,\cdots,n \\ &\xi_i \geq 0 \end{aligned} \] т»╣тЂХжЌ«жбў: \[ \begin{aligned} max_{\lambda}\ \ &-\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j + \sum_i\lambda_i \\ s.t.\ \ &\sum_i\lambda_i y_i = 0 \\ &C-\lambda_i-\mu_i = 0 \\ &\lambda_i, \mu_i \geq 0 \end{aligned} \] тЁХСИГ \(\lambda_i\) Тў»т»╣жЌ┤жџћт╝ЋтЁЦуџёС╣ўтГљ, \(\mu_i\) Тў»т»╣ТЮЙт╝ЏтЈўжЄЈт╝ЋтЁЦуџёС╣ўтГљ. УДБтЄ║ТюђС╝ўУДБ: \[ \begin{aligned} w_*&=\sum_i\lambda_i y_i x_i \\ b_*&=y_i-\sum_i \lambda_i y_i x_i^Tx_j \end{aligned} \] ТГцтцёуџё \(\lambda_i\) УАеуц║ТюђС╝ўуџё \(\lambda_*\).

Тћ»ТїЂтљЉжЄЈ

т»╣С║јуАгжЌ┤жџћ, Тћ»ТїЂтљЉжЄЈТў»Сй┐ \(\lambda > 0\) уџёТаиТюг, ућ▒С║њУАЦТЮЙт╝ЏТђД, т«ЃС╗гТ╗АУХ│ \(y_i(w^Tx_i+b) = 1\), тЏаТГцУ┐ЎС║ЏТаиТюгУљйтюежЌ┤жџћУЙ╣уЋї, т░єтѕєуд╗УХЁт╣│жЮб"Тћ»ТїЂ"УхиТЮЦ.

т»╣С║јУй»жЌ┤жџћ, Тћ»ТїЂтљЉжЄЈТў» Сй┐ \(0 < \lambda < C\) уџёТаиТюг, ућ▒С║њУАЦТЮЙт╝ЏТђД, ТГцТЌХ \(y_i(w^Tx_i+b) = 1\) СИћ \(\xi_i = 0\); тљїТЌХ, УІЦ \(\lambda = C\) тЇ│ \(\mu = 0\), тЇ│ \(\xi_i \neq 0\):

\(0 < \xi_i < 1\): тѕєу▒╗ТГБуА«

\(\xi_i = 1\): тюетѕєуд╗УХЁт╣│жЮбСИі;

\(\xi_i > 1\): У»»тѕє

тљѕжАхТЇЪтц▒ (hinge loss)

у║┐ТђДТћ»ТїЂтљЉжЄЈТю║уџётјЪтДІС╝ўтїќжЌ«жбў \[ \begin{aligned} min_{w,b}\ \ &\frac{1}{2}||w||^2 + C\sum_i\xi_i \\ s.t.\ \ &y_i(w^Tx_i+b) \geq 1 - \xi_i,\ \ i=1,\cdots,n \\ &\xi_i \geq 0 \end{aligned} \] уГЅС╗иС║јжЌ«жбў \[ min_{w, b}\ \ \sum_i[(1-y_i(w^Tx_i+b))]_++\lambda||w||_2^2 \] тЁХСИГ \[ [z]_+= \left \{ \begin{aligned} z&,\quad z>0 \\ 0&,\quad z\leq 0 \end{aligned} \right . \] уД░СИ║тљѕжАхТЇЪтц▒. тљјжЮбуџёжЌ«жбўтЈ»С╗ЦуљєУДБСИ║: тйЊжЌ┤жџћтцДС║ј1ТЌХ, ТЌаТЇЪтц▒; тйЊжЌ┤жџћт░ЈС║ј1ТЌХу╗ЎТЇЪтц▒; тљїТЌХуггС║їжА╣Тў»тЈѓТЋ░уџёТГБтѕЎжА╣.

т»╣Т»ћТёЪуЪЦТю║, у║┐ТђДSVMУдЂТ▒ѓтѕєу▒╗тцДС║јСИђт«џуџёуй«С┐Ат║дтљј, ТЅЇУЃйт░єТЇЪтц▒У«Йуй«СИ║0, УђїТёЪуЪЦТю║ТЌауй«С┐Ат║д, УДЂСИІтЏЙ

Kernel Trick

С╗ју║┐ТђДSVMСИЇУЃйУДБтє│жЮъу║┐ТђДжЌ«жбўуџёУДњт║дтЄ║тЈЉ, т╝ЋтЁЦС║єТаИТіђтиД, С╗јУђїтЙЌтѕ░жЮъу║┐ТђДSVM.

ТаИТіђтиДуџёТђЮТЃ│Тў»т░єтјЪуЅ╣тЙЂуЕ║жЌ┤у╗ЈУ┐Є жЮъу║┐ТђДТўат░ё, Тўат░ёУЄ│СИђСИфуЅ╣тЙЂуЕ║жЌ┤, Сй┐тЙЌтјЪуЅ╣тЙЂуЕ║жЌ┤СИГуџёУХЁТЏ▓жЮбт»╣т║ћСИјТўат░ётљјуЅ╣тЙЂуЕ║жЌ┤уџёуџёУХЁт╣│жЮб. У«░жЮъу║┐ТђДТўат░ёСИ║ \(\phi\). С║Іт«ъСИі, У»ЦжЮъу║┐ТђДТўат░ёт░єУЙЊтЁЦуЕ║жЌ┤ (ТгДт╝ЈуЕ║жЌ┤Тѕќуд╗ТЋБуЕ║жЌ┤) Тўат░ёУЄ│СИђСИфуЅ╣тЙЂуЕ║жЌ┤ (HilbertуЕ║жЌ┤). тЇ│: \[ \phi(x):X \rightarrow H \] СйєТў», ућ▒С║ј \(\phi\) СИЇтЦйТъёжђа, УђїуЏ┤ТјЦУ«Ау«ЌтєЁуД» \(\phi(x)^T\phi(y)\) Т»ћтЁѕУ«Ау«Ќ \(\phi(x)\) тєЇУ«Ау«ЌтєЁуД»т«╣ТўЊ, тЏаТГцт╝ЋтЁЦ ТаИтЄйТЋ░ уџёТдѓт┐х:

ТаИтЄйТЋ░

уД░ \(K(x,z)\) СИ║ТаИтЄйТЋ░, тдѓТъюТ╗АУХ│: \(\forall x,z\in X, \exists \phi(x):X \rightarrow H,\ s.t.\ K(x,z)=\phi(x)\cdot\phi(z)\)

У┐ЎТЌХ, тЈ»С╗Цт░є \[ \begin{aligned} max_{\lambda}\ \ &-\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j + \sum_i\lambda_i \\ s.t.\ \ &\sum_i\lambda_i y_i = 0 \\ &C-\lambda_i-\mu_i = 0 \\ &\lambda_i, \mu_i \geq 0 \end{aligned} \] СИГуџё \[ \frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j x_i^Tx_j - \sum_i\lambda_i \] Тћ╣СИ║ \[ W(\lambda)=\frac{1}{2}\sum_{i,j}\lambda_i\lambda_j y_i y_j K(x_i,x_j) - \sum_i\lambda_i \]
